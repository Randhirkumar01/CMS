Introduction

DCA, Diploma in Computer Applications, is a one-year diploma course in the field of Computer Applicat ions which involves the study of numerous computer applications such as MS Office, Internet Applications, Operating System, Database Management System (DBMS), HTML among other subjects.

After completing a diploma in computer application course, one can either go for further studies in a related field or can also opt for Diploma in Computer Applications jobs in areas like Accounting, Database Handling, Basic Computer Applications, Software Developer, Computer Operator, etc.

Syllabus

The course is dividing into two semesters. Semester 1 for 6-month Duration and Semester 2 for rest 6 month :

Semester 01 –

  • Introduction to Computers
  • Principles of programming
  • Programming Language Database
  • Document handling
  • Word Processing and Spreadsheet
  • System Analysis and Design
  • Computer Graphics
  • Mail merge


Semester 02 –

  • Project Management
  • Communication and Internet
  • WWW and Web browser
  • Unix Operating System
  • Management information System
  • Financial Accounting System
  • C++
